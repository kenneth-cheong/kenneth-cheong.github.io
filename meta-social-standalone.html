Standalone HTML starter

Connect-first Meta dashboard

This version is fully standalone HTML. It can use the Facebook JavaScript SDK in-browser, store session data locally, and call approved endpoints where browser access works. For Instagram, this file includes a manual token workflow because the secure authorization-code exchange normally requires a backend secret.
