Implementation

To integrate this tour into your Campaign Board Crawler:

Step 1 Install Driver.js
npm install driver.js # or use CDN in HTML
Step 2 Add Tour Script
<script src="campaign-board-crawler-tour.js"></script>
Step 3 Launch Tours
// Auto-initializes on page load // Or manually trigger: window.CampaignAuditorTour.startMainTour();
