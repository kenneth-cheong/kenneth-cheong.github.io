Access Levels Table

Role Tier Capabilities & Access
Admin
Full system access. Can view, edit, delete, and import all data across all boards. Manages users and settings.
Vendor
Specifically for external sellers. Can only access "Delivered Backlinks". Can submit/import fulfilled items and request confirmation.
Viewer
Standard view access. Can see all boards and data but cannot edit information or send new backlink requests.
Buyer
View access + requesting permissions. Same as Viewer, but can initiate new requests for backlinks from the Campaigns page.
